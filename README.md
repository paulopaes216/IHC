# Sistema de Login – Banco

## Descrição do Projeto
Este projeto consiste em um sistema de login funcional simulado, desenvolvido para um banco fictício. 
O sistema funciona localmente, sem necessidade de conexão com internet ou banco de dados, com foco na aplicação de conceitos de Interação Humano-Computador (IHC), priorizando usabilidade e experiência do usuário.

---

## Metas de Usabilidade

### Fácil de lembrar como usar
A interface segue o padrão tradicional de sistemas bancários, com campos claros de usuário e senha, facilitando a memorização do uso.

### Fácil de entender
Os rótulos, mensagens de erro e feedback são diretos e escritos em linguagem simples.

### Útil
Permite simular o processo de autenticação de um sistema bancário, cumprindo o objetivo proposto.

### Seguro (percepção do usuário)
O uso de campo do tipo senha e mensagens genéricas de erro aumentam a percepção de segurança.

### Eficiente
O usuário consegue realizar a ação principal (login) com poucos passos e sem distrações.

---

## Metas de Experiência do Usuário

- Sensação de simplicidade
- Confiança no sistema
- Baixa frustração em caso de erro
- Clareza nas ações possíveis
